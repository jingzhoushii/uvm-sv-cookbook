## 📅 4 周学习计划

### 第 1 周：SV 基础
- 数据类型
- 类和 OOP
- 随机化和约束
- 线程和通信

### 第 2 周：UVM 核心
- 阶段机制
- 组件体系
- 事务和序列
- TLM 通信

### 第 3 周：UVM 进阶
- 高级序列
- 寄存器模型
- 中断验证
- 低功耗验证

### 第 4 周：项目实战
- 综合项目
- 测试计划
- 回归测试
