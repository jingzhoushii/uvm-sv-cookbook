# Learning Paths - 三条学习曲线

## 选择你的路线

### 🚀 路线 A：快速上手（0 → 初级验证工程师）
**目标**：应届生 / 转行

| 周 | 内容 | 时间 |
|----|------|------|
| 1-2 | SV 基础（01-sv-fundamentals） | 2 周 |
| 3-4 | UVM 核心（02-06） | 2 周 |
| 5-6 | 基础项目实践 | 2 周 |

**重点**：快速跑通第一个 test

### 🛠️ 路线 B：工程进阶（初中级 → 中高级）
**目标**：1-3 年经验

| 周 | 内容 | 时间 |
|----|------|------|
| 1-2 | 序列高级（07） | 1 周 |
| 3-4 | 寄存器模型（09） | 1 周 |
| 5-6 | 性能优化（13） | 1 周 |
| 7-8 | 形式验证（14） | 1 周 |

**重点**：真实项目经验

### 🏗️ 路线 C：架构设计（中高级 → 骨干）
**目标**：3+ 年经验

| 周 | 内容 | 时间 |
|----|------|------|
| 1-2 | 工业级 Scoreboard | 1 周 |
| 3-4 | VIP 开发 | 2 周 |
| 5-6 | 完整项目 | 3 周 |

**重点**：设计验证平台架构
